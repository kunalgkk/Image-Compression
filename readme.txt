Requirements

To run this project, make sure the following software and libraries are installed:

Python (version 3.8 or higher)
NumPy – for numerical computations
Pandas – for data handling and preprocessing
Scikit-learn – for implementing PCA and K-Means Clustering
Matplotlib – for data visualization
Seaborn (optional) – for enhanced visualizations
Jupyter Notebook or any Python IDE (VS Code, PyCharm, etc.)

Installation
You can install the required libraries using:

```
pip install numpy pandas scikit-learn matplotlib seaborn
```

These tools are necessary to perform dimensionality reduction using PCA and clustering using the K-Means algorithm, as well as to visualize and analyze the results.
